香港是国内用户最关注的机房之一。Evoxt 香港目前使用 **AMD EPYC-Milan 处理器**，三网回程走 **CN2 优化线路**，整体大陆访问体验不错。

具体线路情况：

### 🇯🇵 日本大阪机房（软银线路 / Premium Network）

日本大阪机房接入的是 **软银 BBTEC 线路**，这条线路的特点是解锁日本流媒体能力拔群——**Netflix、Disney+、Amazon Prime、ChatGPT、Gemini 全家桶原生解锁**，IPv4 解锁能力在非住宅IP的机房里属于顶级水准。

### 🇯🇵 日本东京机房（标准线路）

东京机房的情况和大阪有所不同。CPU和磁盘I/O性能表现不俗，但网络线路三网绕路较明显：

- 电信、联通去回程基本绕行美国，延迟偏高（均值约 325ms）

东京机房的核心优势在于**日本原生IP**，解锁日本 Netflix、Disney+ 等本地内容表现出色。适合需要日本IP解锁流媒体的用户，不适合需要低延迟回国的场景。
